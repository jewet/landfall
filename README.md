# Landfall 🛡️
### AI Resilience Passport for Caribbean Buildings

> *Every building should know its own weaknesses — before the storm finds them.*

Built for the **Future Caribbean Global AI Buildathon 2026** | Track: AI for Real Estate & Development

## The problem

After every hurricane, inspectors walk into destroyed neighbourhoods with no record of how those homes were built — no site risk assessment, no compliance history, no construction evidence. Recovery starts almost blind.

The Caribbean loses billions each cycle not just from storms, but from buildings that were in the wrong place, built to the wrong spec, or never properly documented. That's an information problem, not just a weather problem.

## The solution

Landfall is a three-agent AI system that follows a property through its pre-construction and build phases, generating a **Resilience Passport** — a verified record that exists before the storm arrives.

| Agent | What it does |
|-------|-------------|
| **Site Risk Agent** | Pulls flood, hurricane, and terrain data for any Caribbean location and produces a risk score with specific building recommendations |
| **Compliance Agent** | Checks a building description against OECS standards and local hurricane-resilience codes, flagging critical gaps |
| **Passport Generator** | Combines both into a structured, shareable Resilience Passport — an A–F graded record for insurers, government, and owners |

## Who pays for this

- **Government permitting offices** — AI pre-screening of site applications before inspector time is spent
- **Insurance companies** — Risk scoring that reflects actual build quality, not just location
- **Developers** — Catch compliance issues during design, not after inspection failure
- **Homeowners** — A certificate that proves their building was built right

## Tech stack

- **AI**: Anthropic Claude API (`claude-sonnet-4-6`) — powers all three agents
- **Frontend**: Vanilla React via CDN, single-file deployment
- **No backend required** — runs entirely client-side for the demo

## Five-year vision

The passport becomes the standard way a Caribbean building proves itself — the way a vehicle history report became standard for used cars. Governments require it for permitting. Insurers price against it. Property listings include it. And once that record exists, the natural next step is using the same data to run the building — which becomes the full EstateOS platform.
